## 🚀 使用方法

1. 在应用中切换到"烧录"标签页
2. 从预设固件列表中选择固件，或点击"从文件选择"按钮
3. 选中的固件会显示蓝色高亮和勾选标记
4. 外部选择的文件会在底部显示独立卡片
