# 烧录页面修改 - 移除预设固件列表

## 📝 修改内容

### 已移除的功能

1. **预设固件列表**
   - 移除了从 assets 加载的预设固件列表
   - 移除了预设固件的卡片展示
   - 移除了 `presetFirmwaresProvider` 的使用

### 保留的功能

1. **从文件选择固件** ✅
   - 保留了"从文件选择"按钮
   - 保留了文件选择功能
   - 选中后显示美化的固件信息卡片

2. **完整的烧录功能** ✅
   - 保留了烧录进度显示
   - 保留了初始化设置（超时时间、重试次数）
   - 保留了开始/停止/重试按钮
   - 保留了所有烧录逻辑

3. **UI 优化** ✅
   - 未选择文件时显示空状态提示
   - 选中文件后显示大卡片展示
   - 可以取消选择已选文件
   - 保持原有的渐变背景和美化设计

## 🎨 UI 改进

### 固件选择区
- **未选择状态**：显示上传图标和提示文字
- **已选择状态**：显示大卡片，包含：
  - 文件图标（圆形背景）
  - 文件名（粗体显示）
  - 文件大小
  - 版本信息（如果有）
  - 取消选择按钮

### 其他区域
- 烧录进度区：保持不变
- 初始化设置区：保持不变
- 操作按钮区：保持不变

## 🚀 使用流程

1. 打开烧录页面
2. 点击"从文件选择"按钮
3. 选择 HEX 固件文件
4. 查看选中的文件信息
5. 调整初始化设置（可选）
6. 点击"开始烧录"按钮
7. 查看烧录进度

## 🔄 主要变化

### 之前
- 显示预设固件列表（从 assets 加载）
- 可以从列表中选择预设固件

### 现在

- 移除了预设固件列表
